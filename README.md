--

## Install

1. Run the following in the local terminal

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

2. Include Cargo's bin directory. Configure the current shell:

```bash
chmod 777 $HOME/.cargo/env
source $HOME/.cargo/env
```
